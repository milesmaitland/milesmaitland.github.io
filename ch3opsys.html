Term Definition
Operating System An operating system (OS) is system software that manages computer hardware and software resources and provides various services for computer programs.
Job Queue A job queue is a sequence of processes waiting to be executed by the system.
FIFO FIFO (First-In-First-Out) is a scheduling algorithm that processes tasks in the order they arrive, with the earliest arriving task being served first.
Interactive Processing Interactive processing involves real-time interaction between a user and the computer system, where the user provides input and receives immediate responses.
Time-Sharing Time-sharing is an operating system feature that allows multiple users to share a single computer simultaneously, each having the illusion of being the sole user.
Multiprogramming Multiprogramming is a technique where multiple programs are loaded into memory and executed concurrently, allowing efficient utilization of the CPU.
Application Software Application software is a type of software designed to perform specific tasks or applications for users, such as word processing or web browsing.
System Software System software is software that provides a platform for running application software and manages computer hardware.
Utility Software Utility software is a type of system software designed to assist in analyzing, configuring, optimizing, and maintaining the computer system.
Graphical User Interface (GUI) A graphical user interface (GUI) is a visual interface that allows users to interact with computers using graphical elements such as icons, buttons, and windows.
Shell A shell is a command-line interface that allows users to interact with the operating system by entering commands.
Kernel The kernel is the core component of an operating system that manages system resources and provides the lowest-level interface for hardware and software.
Device Driver A device driver is software that allows the operating system to communicate with and control hardware devices.
File Manager A file manager is software that allows users to navigate, organize, and manage files and directories on a computer system.
Memory Manager A memory manager is a component of the operating system that manages the computer's memory, including allocation and deallocation of memory for processes.
Virtual Memory Virtual memory is a memory management technique that uses a combination of hardware and software to provide the illusion of a larger main memory by using secondary storage.
Bootstrapping Bootstrapping is the process of starting a computer by loading the operating system into memory and initializing system components.
ROM Read-Only Memory (ROM) is a type of non-volatile memory that contains permanent or semi-permanent data and is used for booting and essential system functions.
Deadlock Deadlock is a situation in computing where two or more processes are unable to proceed because each is waiting for the other to release a resource.
Spooling Spooling (Simultaneous Peripheral Operation On-Line) is a technique used to queue and manage print jobs to allow for efficient printing without affecting other processes.
Super User (or Administrator) A super user, often referred to as an administrator, is a user with elevated privileges and access rights to manage and configure a system.
Multi-factor Authentication Multi-factor authentication (MFA) is a security process that requires users to provide two or more authentication factors before gaining access to a system or application.
