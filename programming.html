Programming Structures

  • Sequential sturucture: Top-down execution
  • Decision structures: if/then/else
  • Iterative (looping): while loop: continue looping until something is true repeat/until loop: continue looping until something is true. for loop: a given number of times
  • Recusive structures: Repeat a set of instructions as a subtask of itself Repetition continues unti: degenerative case, which is a termination condition, is met (degenerative case is also called a "base case")
